# Worked Examples — Information Architecture Skill

Two finished IA deliverables generated by this skill, one per business type. Open the `.xlsx`
files (two sheets each: **IA Map** + **Vetting Log**); the `.csv` files mirror each sheet for
pasting into Google Sheets.

## 🛒 eCommerce — Koala Living (koalaliving.com.au)
- **`ia-map-koalaliving-com-au-2026-06-05.xlsx`** + CSVs
- 2,956 product PDPs in the sitemap (40 sampled), store-locator pages mapped to **Locations**, and brand collections mapped to **Categories**.
- **Key finding:** the site has almost no category landing pages, so high-volume category terms are flagged as **gaps to build** — Sofas/Lounges (74k), Beds (60k), Mattresses (60k), Rugs (74k), Dining Tables (49k), Coffee Tables (74k). This is the classic eCommerce IA opportunity.

## 🛠 Service — StudioHawk (studiohawk.com.au)
- **`ia-map-studiohawk-com-au-2026-06-05.xlsx`** + CSVs
- Services, industry/sector pages, location pages, case studies, blog and about — all classified, with real SE Ranking volumes on the money pages (e.g. content-marketing-services 2,900; home/seo 1,500; SEO Brisbane 2,400).
- Shows how a service site's IA splits into Services × Industries × Locations.

## How these were made
`siteone`/sitemap crawl → URL classification into sections → **sampled** SE Ranking keyword data on the hub/category/service pages (long-tail PDPs/posts left as TBC) → auto-vetting (KEEP/REMOVED + reason) → `build_ia_csv.py`. Numbers are real SE Ranking pulls; nothing is fabricated.
